# wisun

## How to use

1. Get PaC Device(Client) Info
    - `getVer.py`, `getInfo.py`, `getRegister.py`
2. Get PAA Device(Server) Info
    - `scan.py`
3. Set Config
    - `config.py`, `setConfig.py`
4. Connect/Disconnect to PAA(like server) as PaC(like client)
    - `connect.py`, `disconnect.py`
5. Get Data from PAA
    - `loopGet.py`
